
# ngx-avatar

Run `npm i ngx-avatar` for installing.

Import and add InputModule in your app module.

`import { ProfileImageModule } from 'ngx-avatar';`

Declare as

`<app-profile-image
  [first_name]="Jaya"
  [last_name]="Krishna"
  >
</app-profile-image>`

Declare as for small size

`<app-profile-image
  [first_name]="Jaya"
  [last_name]="Krishna"
  [small]="true"
  >
</app-profile-image>`
